Date and Time Data Types
========================

Syntax
------

===========  =============================================  =============
 Type         Format                                         Description
===========  =============================================  =============
 date         'YYYY-MM-dd',
              'yy-MM-dd',
              'YYYYMMDD',
              'yyMMdd',
              YYYYMMdd,
              yyMMdd,
-----------  ---------------------------------------------  -------------
 datetime     'YYYY-MM-DD hh:mm:ss[.fraction]'
-----------  ---------------------------------------------  -------------
 time         'hh:mm:ss[.fraction]'
              'hhmmss'
-----------  ---------------------------------------------  -------------
 timestamp    'YYYY-MM-DD hh:mm:ss[.fraction][time zone]'
===========  =============================================  =============
